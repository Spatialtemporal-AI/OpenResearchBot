# Report Schema (Research Assistant)

The report must be Markdown and include:

1. **Summary**
2. **Inferred Domain & Task** (include confidence and uncertainties)
3. **Assumptions / Setup** (what can and cannot be inferred from video)
4. **Timeline of Key Events** (with timestamps)
5. **Key Observations** (domain-specific)
6. **Failure / Risk Analysis** (if applicable; include evidence)
7. **Actionable Recommendations**
   - Data collection / labeling
   - Model / policy changes
   - Prompting / instruction changes
   - Method / system changes specific to inferred domain
   - Evaluation metrics and ablations
8. **Next Experiment Plan** (3-6 items with pass/fail criteria)
9. **Appendix**
   - Video metadata summary
   - Model parameters
   - Any notable limitations
